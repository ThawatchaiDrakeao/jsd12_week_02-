sql ภาษาในการจัดการข้อมูล DataBase  จะดึงข้อมูลออกมาทำเป็นตารางโดยใช้คำสั่ง ต่างๆ

เช่น 

 SELECT description, price
FROM MenuItems
ORDER BY price ASC
LIMIT 5;


SELECT = คือการเลือก คอลัม ที่อยากจะให้คำสั่งทำงาน 
ตย. description, price 
คำอธิบาย ราคา

FROM = เลือก"ตาราง" ที่จะให้คำสั่งสั่งนำมาใช้ 
ตย. MenuItems
เมนู

ORDER BY = คือ การใส่คำสั่งที่เราต้องการวางเงื่อนไข จะเรียงข้อมูลยังไง 
ตย. price ASC 
เรียง ราคา  จากน้อยไปมาก

LIMIT ; = คือ การเลือกคำสั่งจำนวน 
ตย. LIMIT 5;  เลือกมา 5 รายการ
